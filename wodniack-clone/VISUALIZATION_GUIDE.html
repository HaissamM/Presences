🎯 Comment visualiser le site

⚠️ Note importante: En raison des contraintes du système, nous ne pouvons pas démarrer un serveur de développement en arrière-plan. Voici les alternatives:
1 Option 1: Utiliser un terminal local

Ouvrez un terminal dans le dossier du projet et exécutez:

cd /vercel/sandbox/wodniack-clone
npm run dev

Puis ouvrez votre navigateur à l'adresse: http://localhost:3000

2 Option 2: Build de production

Le projet a déjà été construit. Pour le servir:

cd /vercel/sandbox/wodniack-clone
npx serve .next -l 3000
3 Option 3: Déploiement

Déployez le site sur Vercel pour une visualisation en ligne:

cd /vercel/sandbox/wodniack-clone
vercel deploy

🛠️ Commandes disponibles

Développement

Production

Qualité du code
